```c#
using System;
using System.Net.Http;
using System.Text;
using System.Threading.Tasks;

class Program
{
  private static readonly HttpClient client = new HttpClient();

  static async Task Main(string[] args)
  {
    var payload = new StringContent(
        "{\"prompt\": \"Cinematic Portrait, GodlyBeautiful french supermodel, dynamic lighting, [light + space of James Turrell + Bauhaus architectural forms], BeautyCore, Sharp Details --ar 21:9 --style raw\"}",
        Encoding.UTF8,
        "application/json"
    );

    client.DefaultRequestHeaders.Add("Authorization", "Bearer <TokenID>");

    var response = await client.PostAsync("http://example.com/items/images/", payload);

    var responseString = await response.Content.ReadAsStringAsync();

    Console.WriteLine(responseString);
  }
}
```

## 什么是 ImagineAPI.dev？

ImagineAPI.dev 是 Midjourney 的非官方 REST API。换句话说，它是一种以编程方式访问和自动化 Midjourney 的方法。您可以使用任何编程语言和标准的 REST 方法（如 `GET` 和 `POST`）来生成图片。

您可以使用它来创建任何您想象的东西——从您的下一个创业公司到为您的朋友和家人创建的简单机器人。
