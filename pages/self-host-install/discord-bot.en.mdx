# Create a Discord Server and Bot

<Callout type="info" emoji="🏎️">
If this looks like a lot of work, we do all this for you and more on our [Cloud Plans](https://www.imagineapi.dev/pricing).
</Callout>

Follow these steps to create a discord server, invite `Midjourney bot` to it and create your own bot. These are required for ImagineAPI.

## Create a Discord server

<Callout>
If you already have a Discord server where you have invited `Midjourney Bot` you don't need to skip this part.
</Callout>

1. Create a Discord server: <Image src="/CleanShot 2023-04-11 at 15.58.07@2x.png" width={800} height={500} />
1. Name it whatever you would like.
1. Let's now invite `Midjourney Bot` into your new server.
1. Go to the `Midjourney` server and click on `Midjourney Bot`: <Image src="/CleanShot 2023-04-11 at 16.01.58@2x.png" width={800} height={500} />
1. Click `Add to Server`: <Image src="/CleanShot 2023-04-11 at 16.03.44@2x.png" width={800} height={500} />
1. Select the Server you just created to add Midjourney Bot to it: <Image src="/CleanShot 2023-04-11 at 16.05.35@2x.png" width={800} height={500} />
1. Click `Authorize`: <Image src="/CleanShot 2023-04-11 at 16.07.04@2x.png" width={800} height={500} />


## Create a Discord Bot

1. Go to https://discord.com/developers/applications
1. Create a new application (give it a random name..it doesn't matter) <Image src="/CleanShot 2023-05-03 at 14.24.26@2x.png" width={800} height={500} />
1. Go to `OAuth2 > URL Generator` page, select `bot` under Scopes.<br />Then select `Manage Channels`, `Read Messages/View Channels`, `Use Slash Commands`, and `Send Messages` under `Bot Permissions` <Image src="/CleanShot 2023-05-03 at 14.26.04@2x.png" width={800} height={500} />
1. Click Copy to get the URL and go to that URL.
1. Under `Add to Server` select your [newly created server](#create-a-discord-bot): <Image src="/CleanShot 2023-05-03 at 14.52.03@2x.png" width={800} height={500} />
1. Authorize the permissions: <Image src="/CleanShot 2023-05-03 at 14.51.29@2x.png" width={800} height={500} />
1. Go back to the Discord Developer Portal and go to the Bot page and click `Add Bot`
1. Turn on `Message Content Intent`: <Image src="/CleanShot 2023-04-12 at 02.30.01.png" width={800} height={500} />
1. Save the changes.
1. Then copy the Token: <Image src="/CleanShot 2023-04-11 at 15.38.12@2x.png" width={800} height={500} />
1. Keep this token around as you will need it soon.
