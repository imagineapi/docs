# 升级 ImagineAPI

<Steps>

### 从 Github 获取最新代码
运行以下命令获取 ImagineAPI.dev 的最新版本：

```sh copy
git pull
```

<Callout>
`git pull` 大多数情况下不会包含任何更改。无论如何，请继续执行后面的更新步骤。
</Callout>

### 获取 Docker 镜像更新

获取最新的 `docker-compose.yml` 和相关文件。

然后拉取新的 Docker 镜像：

```sh copy
docker compose pull
```

您将看到有更新的 Docker 镜像被下载：

```
$ docker compose pull
Pulling database ... done
Pulling rabbitmq ... done
Pulling api      ... done
Pulling bot      ... done
Pulling autoheal ... done
```

### 使用新镜像重新创建 Docker 容器

然后启动有更新的容器：

```sh copy
docker compose up -d
```

</Steps>
