## What is ImagineAPI?

An **unofficial** REST API for Midjourney. You can programmatically generate images using any programming language and standard REST methods like `GET` and `POST`.

<Image src="/b36e9627-c3de-4f25-82c5-196b4859494b.png" alt="steampunk city with airship in background zoomed out to fit everything in shot stormy sky:: --ar 2:1 --v 4 --quality 2 --v 5 --s 50" width={850} height={500} />

You can use it to create any thing you can imagine â€” from your next startup to a simple bot for your friends and family.

Here's a quick Python example to generate the above image:

```py filename="main.py"
import requests
import consts # this has the API key and URL

# Define the prompt to send to Midjourney 
data = {'prompt': 'oil spill flamingo'}

# Send the POST request
response = requests.post(consts.base_url+"/images", json=data, headers=consts.auth)

# Print the response status code and content
print('Status code:', response.status_code)
print('Response content:', response.content)
```
