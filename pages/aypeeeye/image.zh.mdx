# 图像

提供所有方法来提供一个[验证令牌](/API/authentication)。

## `GET /items/images/:id`

返回： 

```json
{
    "data": {
        "id": UUID,
        "prompt": 字符串,
        "results": 无 | 字符串,
        "user_created": 字符串,
        "date_created": 字符串,
        "status": "pending" | "in-progress" | "completed" | "failed",
        "progress": 数字 | 无, 
        "url": 无 | 字符串,
        "error": 无 | 字符串,
        "upscaled_urls": 无 | 字符串[],
        "upscaled": 字符串[]
    }
}
```
