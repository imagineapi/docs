# Authentication

Each request needs to provide an authentication token in the headers:

`Authorization: Bearer <Token>`

You can get the token from the API admin by clicking on your user (login here first: [cl.imagineapi.dev](https://cl.imagineapi.dev/)):

<Image src="/CleanShot 2023-04-12 at 01.41.51.png" width={800} height={500} />

Then scroll to `Admin Options` and clikc `+` in the `Token` field:

<Image src="/CleanShot 2023-04-12 at 01.42.02.png" width={800} height={500} />

Copy the token and click the check mark:

<Image src="/CleanShot 2023-04-12 at 01.42.09.png" width={800} height={500} />
